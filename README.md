### Introduce  
 - #### 주요 기능
   - **채팅🗣🗯💬**</br>
  소켓통신을 이용한 실시간 채팅. 개인 대화, 단체 대화

   - **영상 통화🎥📞**</br>
 **`WebRTC`** 를 이용한 실시간 영상통화. 1:1 영상 통화  

   - **이미지 전송🌃🎆🌆**</br>
 기기 갤러리 안의 이미지 파일을 실시간 채팅 중 전송

   - **친구 추가,삭제 👥**</br>
 특정 유저를 친구 목록에 추가, 삭제하여 관리가능
